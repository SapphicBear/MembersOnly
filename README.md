# MembersOnly

A website where you can login and leave a message!

Made as a part of the [Odin Project's Node backend programming course!](https://www.theodinproject.com/paths/full-stack-javascript/courses/nodejs)

**[Check it out here!](https://membersonly-production-522d.up.railway.app)**

## Technologies

This project is using passportjs for user authentication, express-session for user-sessions and user information is saved on a Postgresql server!

This app was created using express.js and EJS!

**Please do NOT** put any sensitive data on the website.

To see usernames, become a member! It's a bit of a puzzel to become one! :D

*This project uses no gen AI at all. No Gen AI was used in the creation or design on this project.*
